# Portfolio

My projects porfolio website.

### Built with:

- Gatsby.js
- Styled Components

## Development

  1. Clone the project on the local machine:
  
  ```
   git clone https://github.com/mariusjagminas/Portfolio.git
  ```

  2. Install all dependencies:

  ``` 
  npm install
  ```

  3. Start the development server:
  ```
  gatsby develop
  ```
